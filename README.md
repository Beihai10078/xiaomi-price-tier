## 功能特性

### 数据规模
覆盖小米、红米、POCO 三大品牌共计 **245 台**已发布设备，从初代小米手机 1 到最新的小米 16 系列、Redmi Note 15 系列、POCO F8 系列等旗舰与中端机型，每一台均包含设备名称、内部研发代号、发布价格区间、搭载的 SoC 芯片型号四大核心字段，数据持续更新维护，是米粉搞机、购机对比、ROM 开发者的实用参考手册。

### 智能搜索
顶部搜索栏支持**模糊实时搜索**，输入设备名称、代号或型号关键词即可瞬间定位。无论是搜"小米 14 Ultra"、"shennong"还是"K70"，都能在 245 台设备中毫秒级筛选出匹配结果。搜索框自带清除按钮和 placeholder 提示，输入体验流畅无卡顿。

### 多维筛选系统
提供**品牌 × 系列 × 类型**三维交叉筛选能力：

| 筛选维度 | 可选值 | 说明 |
|---------|-------|------|
| 品牌 | 全部 / 小米 / 红米 / POCO | 按品牌归属过滤 |
| 系列 | 全部 / 数字系列 / Ultra / Mix / Civi / T / Note / K / Turbo / F / X / M / A / Go 等 20+ 系列 | 按产品线细分 |
| 类型 | 全部 / 手机 / 平板 / 笔记本 / 可穿戴 / 耳机 / 电视 / 路由器 / 智能家居 | 按设备品类过滤 |

桌面端三个筛选栏并排展示，品牌栏与类型栏之间以精致**分栏竖线**分隔（仿 Word/WPS 分栏排版），视觉层次清晰分明。移动端筛选项以 pill 胶囊按钮形式横向排列，选中态高亮，适配触屏操作。三个维度可任意组合，筛选结果实时刷新，卡片数量动态更新。

### 代号一键复制
每张设备卡片右侧展示设备**内部研发代号**（Codename），点击代号旁的复制图标即可一键拷贝至剪贴板，复制成功后按钮短暂变为绿色对勾给予反馈。对于 ROM 开发者、刷机用户来说，从此告别手动抄写代号的痛苦——像 "zeus"、"ishtar"、"marble"、"cupid" 这些代号，点一下就能带走。

### 深色 / 浅色主题
内置精心调校的 **Material You 风格双主题**：

- **浅色主题**：明亮清爽，适合日间办公环境，卡片采用白色基底搭配淡色描边，文字对比度舒适不刺眼
- **深色主题**：深邃护眼，适合夜间/暗光环境浏览，底色采用深灰黑搭配低亮度文字，减少屏幕眩光对眼睛的刺激

主题切换按钮位于页面右上角工具区，桌面端为太阳/月亮图标按钮，移动端为太阳/月亮 SVG +「主题」文字标签的触控按钮。切换时无闪烁、无白屏，主题状态由浏览器 `localStorage` 持久化，刷新/重开页面后自动恢复上次选择的主题，无需重新设置。

### 中英双语切换
全站支持**中文简体 / English** 一键切换，覆盖范围不仅仅是界面文字：

- **界面文案**：搜索 placeholder、筛选标签、统计栏（"245台设备 / 107小米 / 136红米 / 2POCO"）、按钮文字全部双语化
- **设备数据**：设备名称保持中英原名不变（如"小米 14 Ultra"在两种语言下均显示原名），类型/系列/品牌标签同步翻译
- **口号系统**：页面顶部随机轮播的 12 条赛博朋克风格数码口号（如「Fork the Revolution」「Bootloader Unlocked」「数字三民主义」「MIUI 复兴」等），中英文各独立维护 12 条，语言切换时自动对应语种口号库
- **副标题**：页面副标题 "Xiaomi / Redmi / POCO 价格天梯" 同步切换为英文 "Xiaomi / Redmi / POCO Price Tier — Device Codename & SoC Quick Reference"
- **关于弹窗**：弹窗内介绍文字、开发者信息、版本号均双语化

语言偏好同样通过 `localStorage` 持久化，下次访问自动载入首选语言。

### 关于弹窗
点击右上角信息图标或移动端「关于」按钮，弹出**毛玻璃（Frosted Glass）半透明遮罩层**，居中卡片展示项目简介、技术栈说明、作者信息（小红书 + GitHub 入口同列排列，点击直达个人主页与仓库）、当前版本号。弹窗支持点击遮罩区域或关闭按钮关闭，出场动画平滑流畅。

### 返回顶部
右下角悬浮**圆形返回顶部按钮**，并非传统的「滚动超过 N 像素就显示」，而是采用更智能的**双条件触发机制**：仅在页面有滚动偏移 **且** 当前筛选结果卡片数 ≥ 5 时显示。短列表无需返回顶部时不打扰视线，长列表时自然出现，点击后平滑滚动至页面顶部，按钮有 hover 缩放微动效。

### 移动端专属优化
针对手机浏览器进行了大量专项打磨：

- **品牌标识**：移动端顶部使用高清晰度「XIAOMI」文字 PNG 作为品牌标识（3x 放大裁剪至 1266×360），搭配 240px 固定宽度 + `max-width: 100%` 自适应，桌面端则使用完整小米 Logo，双端视觉统一而不失辨识度
- **按钮图标化**：右上角三个工具按钮（主题/语言/关于）在移动端均配有 SVG 图标 + 中文/英文文字标签，太阳/月亮图标、地球图标、信息图标清晰可辨，442px 以下屏幕触发图+文模式，确保触控目标区域足够大
- **统计栏全宽**：底部统计栏（设备总数 / 小米 / 红米 / POCO 四枚计数 chip）在移动端自动均分横向空间，禁止换行，四个 chip 等宽居中排列
- **筛选 pill 全宽**：移动端筛选项 pill 按钮组撑满容器宽度，触控友好，选中态以主题色高亮区分
- **副标题优化**：副标题文字禁止选中（`user-select: none`），避免移动端长按误触选中；间距经过多轮微调至最佳状态

### 响应式布局
基于 **CSS Grid + Flexbox** 构建，卡片网格自动适配屏幕宽度：桌面端 4 列、平板 3 列、大屏手机 2 列、小屏手机 1 列，断点设置合理，从 320px 宽的老手机到 4K 显示器均能完美呈现。筛选栏、统计栏、工具按钮区在不同宽度下自动切换行/列布局。

### 技术架构
**纯静态 HTML 单文件**，零外部依赖，零构建工具，零框架，零 npm 包，零 CDN 引用。所有 CSS 内联，所有 JavaScript 原生手写，所有 SVG 图标内嵌。下载单个 `.html` 文件双击即可在任意现代浏览器中运行——Chrome / Edge / Firefox / Safari 全兼容，甚至可以直接放进 U 盘带到离线电脑上使用。文件体积控制在合理范围（含 245 台设备完整数据），GitHub Pages 一键部署，加载速度极快。

### 设计语言
整体采用 **Material You（Material Design 3）** 设计哲学：卡片式布局、圆角 pill 按钮、柔和阴影与描边、动态主题色系统（CSS 自定义属性驱动，深色/浅色模式下自动切换色板）、微妙的 hover 过渡动效（`cubic-bezier(0.4, 0, 0.2, 1)` 缓动曲线）、毛玻璃遮罩层。配色灵感源自小米品牌橙色（`#ff6900`）为主色调，搭配中性灰色阶构建完整色彩体系。
