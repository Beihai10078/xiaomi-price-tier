# 米系设备价格与代号查询

Xiaomi / Redmi / POCO 设备价格天梯 & 代号速查工具。

## 在线访问

[https://beihai10078.github.io/xiaomi-price-tier/](https://beihai10078.github.io/xiaomi-price-tier/)

## 版本迭代

| 版本 | 文件 | 说明 |
|------|------|------|
| V1 | `xiaomi-price-tier.html` | 基础版本：设备列表、搜索、品牌筛选 |
| V2 | `xiaomi-price-tier-v2.html` | 新增系列筛选、价格展示 |
| V3 | `xiaomi-price-tier-v3.html` | 新增代号复制、类型筛选 |
| V4 | `xiaomi-price-tier-v4.html` | UI 优化、深色主题 |
| V5 | `xiaomi-price-tier-v5.html` | 中英双语切换、12条太阳光芒 |
| V6 | `xiaomi-price-tier-v6.html` | 关于弹窗、毛玻璃遮罩、按钮组优化 |

## 功能特性

- 245 台小米 / 红米 / POCO 设备数据
- 设备名称、代号、价格、SoC 信息查询
- 品牌 / 系列 / 类型多维筛选
- 代号一键复制
- 深色 / 浅色主题切换
- 中英双语切换
- 纯静态 HTML，零依赖，开箱即用

## 本地使用

直接下载任意版本的 `.html` 文件，用浏览器打开即可。

```
双击 xiaomi-price-tier-v6.html → 浏览器中运行
```
